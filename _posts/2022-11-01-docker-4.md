`-f` 옵션을 사용하여 사용자 지정 파일을 지정할 수 있다.

### 서비스 올리기 
```bash
$ docker-compose -f docker-compose-custom.yml up -d
```

### 서비스 내리기
```bash
$ docker-compose -f docker-compose-custom.yml down
```
