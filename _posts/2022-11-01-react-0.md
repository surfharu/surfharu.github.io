`Create-React-App` 을 이용하여 프로젝트 생성해 본다.
> **CRA(Create-React-App) ?**  
> 리액트 프로젝트를 시작하는데 필요한 개발 환경을 자동으로 세팅 해주는 도구이다.

### 프로젝트 생성
```bash
npx create-react-app [new project name]
```

### 모듈 설치
```bash
yarn
 ```

### 실행
```bash
yarn start
 ```
